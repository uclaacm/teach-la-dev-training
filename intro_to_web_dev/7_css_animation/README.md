# CSS Animation

Modern websites often have ~ smooth ~ and fany animations to spice up the page and bring some life to it. These are achieved primarily with CSS animations and JS. Though these days, developers use libraries and special software and pipelines to make very advanced and complicated animations easily and quickly, the base of it is still CSS animation and JS.
